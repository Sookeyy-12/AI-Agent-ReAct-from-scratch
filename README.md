# AI Agent ReAct from Scratch 🤖🚀

A powerful, lightweight implementation of a **Reason and Action (ReAct)** agent built using Python and the Google Gemini API. This project demonstrates how to build an autonomous agent that can reason through complex queries, use multiple tools in parallel, and maintain context over a conversation.

## 🌟 Key Features

- **Autonomous execution loop**: The agent recursively processes reasoning and tool calls until it reaches a final answer.
- **Parallel Tool Calling**: Efficiently handles multiple tool requests in a single turn.
- **Thought -> Action Pattern**: Follows a strict reasoning-first approach to ensure transparency in AI decision-making.
- **Custom Tool Registry**: Easily extensible system for adding mathematical, data-fetching, or API-based tools.
- **Persistent Context**: Full conversation history management for context-aware multi-turn interactions.

## 🧠 How it Works

The agent operates on a continuous feedback loop:
1. **Thought**: The model analyzes the user's prompt and explains its reasoning.
2. **Action**: If needed, the model selects one or more tools to execute.
3. **Observation**: tool execution results are fed back into the model's memory.
4. **Final Answer**: Once the model has sufficient information, it delivers the result to the user.

## 🚀 Quick Start

### 1. Installation
Ensure you have `uv` installed, then clone the repo and sync dependencies:
```bash
git clone https://github.com/Sookeyy-12/AI-Agent-ReAct-from-scratch.git
cd AI-Agent-ReAct-from-scratch
uv sync
```

### 2. Configuration
Create a `.env` file (or set your environment variables) with your Google Cloud project details:
```env
GOOGLE_API_KEY=your_api_key
PROJECT_ID=your_project_id
LOCATION=us-central1
```

### 3. Run the Agent
Start the interactive REPL:
```bash
uv run python main.py
```
